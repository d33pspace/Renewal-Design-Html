📱 About This App

This is a mobile-responsive tea e-commerce application converted from Figma designs to plain HTML, CSS, and JavaScript.

✨ Features:

  • Product browsing and catalog
  • Shopping cart with quantity controls
  • Order checkout and settlement
  • User account management
  • Settings and preferences
  • Responsive mobile design

🛠️ Tech Stack:

  • HTML5 - Semantic markup
  • CSS3 - Modern styling with Flexbox & Grid
  • Vanilla JavaScript - Interactive features
  • Google Fonts - Noto Sans & Roboto Slab
